About Dashboard

A complete guide explaining every LC Dashboard feature and how to use it.

Recommended Workflow

Use this dashboard as a step-by-step LC operations workspace from LC registration to document checking, compliance review, discrepancy tracking and management reporting.

Create LC: Start by creating a new LC case with applicant, beneficiary, issuing bank, amount, currency and expiry details.
Upload Documents: Select the LC case, then drag and drop files, click the upload zone, or use the Browse Files button to attach commercial documents.
Check Compliance: Use the compliance area to review document readiness and LC terms before approval.
Resolve Discrepancies: Track mismatches, missing details and corrections before final processing.
Assign Tasks: Create tasks for operations team members with due dates and priorities.
Monitor KPIs: Review alerts, SLA breaches, exposure, deadlines and reports from the dashboard.

Dashboard

Shows an executive overview of your LC portfolio including total cases, pending reviews, approvals, discrepancies, total exposure, expiring LCs, SLA status, priority alerts and workflow progress.

  • Use KPI cards for quick management review.
  • Use alerts to identify urgent cases.
  • Use charts to understand status and completion trends.

LC Cases

Stores the master LC register. Each case includes LC number, applicant, beneficiary, issuing/advising bank, amount, currency, expiry date and processing status.

  • Create new cases from the Create New LC button.
  • Review all existing cases in table format.
  • Delete incorrect test records if needed.

Document Upload

Attach trade documents to a selected LC case. The drag-and-drop area now supports selecting files by clicking or dropping files directly onto the upload box.

  • Select an LC case first, or upload as Unassigned and organise later.
  • Drop one or many files into the zone.
  • Uploaded file metadata is saved locally in the browser.

Compliance Checker

Designed to help operations teams review whether documents and LC details meet required terms, internal controls and checking standards.

  • Select an LC case.
  • Review compliance result rows.
  • Use this before approval or payment processing.

Discrepancies

Tracks exceptions such as missing documents, late presentation, data mismatch, amount differences, incorrect consignee details or other document inconsistencies.

  • Filter discrepancies by severity.
  • Focus on critical and major issues first.
  • Use this as an exception resolution register.

Task Management

Allows teams to assign operational work related to LC checking, customer follow-up, discrepancy resolution and approvals.

  • Create tasks with owner, priority and due date.
  • Link a task to an LC case.
  • Track status from To Do to Completed.

Calendar

Highlights important LC dates like expiry, shipment deadline, document presentation period and operational follow-up events.

  • Use it to prevent missed deadlines.
  • Review days remaining for each event.
  • Prioritise cases with short time remaining.

Reports

Provides performance and management reporting such as processing rate, revenue, average processing time and variance against target.

  • Use reports for management review.
  • Track operation efficiency.
  • Identify process bottlenecks.

Audit Trail

Records major user actions including created LC cases, tasks and deleted items. This supports traceability and operational control.

  • Filter by action type.
  • Review who did what and when.
  • Use it for internal audit checks.

Settings

Contains user profile and notification preferences. It is useful for configuring user identity and system behaviour.

  • Update user name and email.
  • Toggle notification preferences.
  • Review basic security preferences.

Help Icons

Every major feature has a small question mark icon. Hover over it to see a short explanation of what the feature does.

  • Available on menu items, cards, buttons, headings and labels.
  • Works with mouse hover and keyboard focus.
  • Useful for new users and training.

Global Search

The search box filters visible table rows by LC number, beneficiary, applicant, bank, status and other visible text.

  • Type keywords into the header search box.
  • Matching rows remain visible and highlighted.
  • Clear the search text to reset the view.

Important Notes

  • This prototype stores information locally in the browser using localStorage.
  • Uploaded file content is not sent to a server; only file metadata is stored for dashboard tracking.
  • For production banking use, server-side storage, authentication, audit controls, encryption and approval workflows should be added.
